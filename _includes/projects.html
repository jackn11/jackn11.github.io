Volunteerism
Volunteerism is a site that I, along with a team of 3 others, made as part of the LyonHacks Hackathon. We saw the need for more accessible ways for people to find good, relevant volunteer opportunities, and the site does just that. The site is made using HTML, CSS, and JavaScript. We used Firebase Authentication for the user accounts and Firebase Cloud Firestore for the user, organization, and and opportunity information storage. We also used the OpenStreetMap API to integrate the map feature into the site.
